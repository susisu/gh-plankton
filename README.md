# gh-plankton
[Sometimes GitHub's default user-icon looks like a plankter...](https://susisu.github.io/gh-plankton)

## Development
### Setup
``` shell
git clone https://github.com/susisu/gh-plankton.git
cd gh-plankton
npm install
```

### Start App
``` shell
npm start
```

### Build
``` shell
npm run build
```
